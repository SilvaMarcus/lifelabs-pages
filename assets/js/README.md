# Organização dos scripts

- `global.js`: ajustes globais pequenos.
- `offers.js`: dados dos produtos/ofertas.
- `categories.js`: dados das categorias e ícones dos filtros.
- `render-offers.js`: monta categorias e cards no HTML.
- `store.js`: busca, filtros, menu mobile e botão voltar ao topo.

Ordem no HTML: `global.js` → `offers.js` → `categories.js` → `render-offers.js` → `store.js`.
